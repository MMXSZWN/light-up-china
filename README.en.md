# Light Up China

An interactive China map visualization application to track and display the provinces you have visited.

## Features

- **Interactive Map**: Uses ECharts to render a China map, visually showing visited and unvisited provinces
- **Data Statistics**: Real-time display of the number of visited and unvisited provinces
- **Province Lists**: Separate lists of visited and unvisited provinces for easy viewing
- **Local Storage**: Uses localStorage to save visit records, ensuring data persistence

## Project Structure

```
.
├── index.html    # Main page file
└── script.js     # Core logic script
```

## Usage Instructions

1. Open the `index.html` file directly in your browser
2. Click on a province on the map to mark it as "Visited"
3. Click again on a visited province to unmark it
4. Statistics and province lists will update automatically
